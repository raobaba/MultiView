## Run Locally

Clone the project

```bash
 https://github.com/Aman9723/money-tail-4785.git
```

Go to the project directory

```bash
  cd Frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

### Prerequisites
React.js , Nodejs , Firebase , Chakra UI , Express , Mongo DB , Redux-Thunk , Api , Render.
